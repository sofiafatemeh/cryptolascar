  <name>News headline collection via NewsAPI and BS4 scraping with 2h cache</name>
  <files>collectors/news.py, tests/test_news.py</files>
  <behavior>
    Cache source: "news"
    Cache symbol: "headlines" (all headlines stored as one JSON list under this symbol)
    Cache TTL: 2 hours

    Collection flow:
    1. Check cache: market_cache WHERE source="news" AND symbol="headlines" AND expires_at > utcnow()
    2. Cache hit: return cached headlines dict immediately
    3. Cache miss:
       a. Fetch NewsAPI headlines (~10) if config.newsapi_key is set
       b. Scrape 4 sites (5 headlines each): CoinDesk, CoinTelegraph, Boursorama, AMF
          - Sleep 1.0s between each domain scrape call
       c. Merge all headlines, deduplicate by URL
       d. Cap total at 35 headlines
       e. Upsert merged list to cache: source="news", symbol="headlines", TTL=2h

    Per headline format (D-10):
    {"title": str, "url": str, "source": str, "published_at": str}

    NewsAPI mapping:
    article -> {title: article["title"], url: article["url"],
                source: article["source"]["name"], published_at: article["publishedAt"]}

    Scraping mapping (use BeautifulSoup4):
    For each site, extract up to 5 items. For scraped items without a publication date:
    published_at = datetime.now(timezone.utc).strftime("%Y-%m-%dT%H:%M:%SZ")

    BS4 scraping strategy per site (use simple tag/class selectors — no JS):
    - CoinDesk (https://www.coindesk.com/): find <a> tags containing <h3>, extract text and href, prepend "https://www.coindesk.com" if relative
    - CoinTelegraph (https://cointelegraph.com/): find <span class="post-card-inline__title"> within <a> tags
    - Boursorama (https://www.boursorama.com/bourse/actualites/): find <a class="c-link"> or article title links in the main content div
    - AMF (https://www.amf-france.org/fr/espace-epargnants/actualites-et-alertes): find <a> tags with article/communiqué titles in the main listing

    Note: BS4 selectors may need adjustment when implemented if site HTML changes — the scraper must not crash on selector failures, just skip that source.

    Graceful degradation:
    - NewsAPI unavailable (no key or HTTP error): newsapi_failed=True, continue with scraped
    - Individual scrape site failure: log warning, continue with other sites
    - All sources fail: return {"headlines": [], "newsapi_failed": True, "scrape_failed": True, "partial": True}
    - Never raise

    Return dict shape:
    {
      "headlines": [
        {"title": str, "url": str, "source": str, "published_at": str},
        ... (max 35 items)
      ],
      "count": int,
      "newsapi_failed": bool,
      "scrape_failed": bool,  # True if ALL scraping failed
      "partial": bool,
      "source_used": "news"
    }

    Test cases:
    - Test 1: Cache hit returns headlines without calling httpx or BS4
    - Test 2: Cache miss fetches NewsAPI + scrapes all 4 sites, writes cache, total ≤ 35
    - Test 3: NewsAPI key missing → newsapi_failed=True, scraped headlines still returned
    - Test 4: All scraping fails → scrape_failed=True, NewsAPI headlines still in result
    - Test 5: collect_news never raises — always returns dict even if all sources fail

    ```python
    from __future__ import annotations
    import json
    import time
    from datetime import datetime, timedelta, timezone
    from typing import Any

    import httpx
    from bs4 import BeautifulSoup

    from config import Config
    from db.cache import get_connection
    from logging_setup import get_logger

    logger = get_logger(__name__)

    CACHE_SOURCE = "news"
    CACHE_SYMBOL = "headlines"
    CACHE_TTL_HOURS = 2
    SCRAPE_SLEEP_SECONDS = 1.0
    MAX_HEADLINES = 35
    MAX_PER_SCRAPED_SOURCE = 5
    NEWSAPI_PAGE_SIZE = 10

    NEWSAPI_URL = "https://newsapi.org/v2/everything"
    SCRAPE_SOURCES = {
        "CoinDesk": "https://www.coindesk.com/",
        "CoinTelegraph": "https://cointelegraph.com/",
        "Boursorama": "https://www.boursorama.com/bourse/actualites/",
        "AMF": "https://www.amf-france.org/fr/espace-epargnants/actualites-et-alertes",
    }


    def _utcnow_iso() -> str:
        return datetime.now(timezone.utc).strftime("%Y-%m-%dT%H:%M:%SZ")

    def _expires_iso(hours: int) -> str:
        return (datetime.now(timezone.utc) + timedelta(hours=hours)).strftime("%Y-%m-%dT%H:%M:%SZ")

    def _get_cached(conn) -> list | None:
        now = _utcnow_iso()
        row = conn.execute(
            "SELECT data_json FROM market_cache WHERE source=? AND symbol=? AND expires_at > ?",
            (CACHE_SOURCE, CACHE_SYMBOL, now)
        ).fetchone()
        return json.loads(row["data_json"]) if row else None

    def _upsert_cache(conn, headlines: list) -> None:
        conn.execute(
            "INSERT OR REPLACE INTO market_cache (source, symbol, data_json, fetched_at, expires_at) VALUES (?,?,?,?,?)",
            (CACHE_SOURCE, CACHE_SYMBOL, json.dumps(headlines), _utcnow_iso(), _expires_iso(CACHE_TTL_HOURS))
        )
        conn.commit()

    def _fetch_newsapi(api_key: str) -> list[dict]:
        """Fetch ~10 headlines from NewsAPI. Returns empty list on failure."""
        params = {
            "q": "ETF OR crypto OR bourse OR CAC",
            "language": "fr",
            "sortBy": "publishedAt",
            "pageSize": NEWSAPI_PAGE_SIZE,
            "apiKey": api_key,
        }
        resp = httpx.get(NEWSAPI_URL, params=params, timeout=10)
        resp.raise_for_status()
        articles = resp.json().get("articles", [])
        return [
            {
                "title": a["title"],
                "url": a["url"],
                "source": a["source"]["name"],
                "published_at": a["publishedAt"],
            }
            for a in articles[:NEWSAPI_PAGE_SIZE]
            if a.get("title") and a.get("url")
        ]

    def _scrape_coindesk(html: bytes, base_url: str) -> list[dict]:
        soup = BeautifulSoup(html, "html.parser")
        results = []
        for a_tag in soup.find_all("a", href=True):
            h_tag = a_tag.find(["h3", "h4"])
            if h_tag and h_tag.get_text(strip=True):
                href = a_tag["href"]
                if href.startswith("/"):
                    href = "https://www.coindesk.com" + href
                results.append({"title": h_tag.get_text(strip=True), "url": href,
                                 "source": "CoinDesk", "published_at": _utcnow_iso()})
            if len(results) >= MAX_PER_SCRAPED_SOURCE:
                break
        return results

    def _scrape_cointelegraph(html: bytes, base_url: str) -> list[dict]:
        soup = BeautifulSoup(html, "html.parser")
        results = []
        for span in soup.find_all("span", class_=lambda c: c and "title" in c):
            parent_a = span.find_parent("a", href=True)
            title = span.get_text(strip=True)
            if parent_a and title:
                href = parent_a["href"]
                if href.startswith("/"):
                    href = "https://cointelegraph.com" + href
                results.append({"title": title, "url": href,
                                 "source": "CoinTelegraph", "published_at": _utcnow_iso()})
            if len(results) >= MAX_PER_SCRAPED_SOURCE:
                break
        return results

    def _scrape_boursorama(html: bytes, base_url: str) -> list[dict]:
        soup = BeautifulSoup(html, "html.parser")
        results = []
        for a_tag in soup.find_all("a", href=True):
            classes = a_tag.get("class", [])
            title = a_tag.get_text(strip=True)
            href = a_tag["href"]
            if title and len(title) > 20 and ("actu" in href or "article" in href or "actualite" in href):
                if href.startswith("/"):
                    href = "https://www.boursorama.com" + href
                results.append({"title": title[:200], "url": href,
                                 "source": "Boursorama", "published_at": _utcnow_iso()})
            if len(results) >= MAX_PER_SCRAPED_SOURCE:
                break
        return results

    def _scrape_amf(html: bytes, base_url: str) -> list[dict]:
        soup = BeautifulSoup(html, "html.parser")
        results = []
        for a_tag in soup.find_all("a", href=True):
            title = a_tag.get_text(strip=True)
            href = a_tag["href"]
            if title and len(title) > 20 and ("amf-france.org" in href or href.startswith("/")):
                if href.startswith("/"):
                    href = "https://www.amf-france.org" + href
                results.append({"title": title[:200], "url": href,
                                 "source": "AMF", "published_at": _utcnow_iso()})
            if len(results) >= MAX_PER_SCRAPED_SOURCE:
                break
        return results

    SCRAPE_PARSERS = {
        "CoinDesk": _scrape_coindesk,
        "CoinTelegraph": _scrape_cointelegraph,
        "Boursorama": _scrape_boursorama,
        "AMF": _scrape_amf,
    }

    def _scrape_all() -> tuple[list[dict], bool]:
        """Scrape all 4 sources. Returns (headlines, all_failed)."""
        all_headlines: list[dict] = []
        any_success = False
        first_request = True

        for source_name, url in SCRAPE_SOURCES.items():
            if not first_request:
                time.sleep(SCRAPE_SLEEP_SECONDS)
            first_request = False
            try:
                resp = httpx.get(url, timeout=15, headers={"User-Agent": "Mozilla/5.0 (compatible; CryptoLascar/1.0)"})
                resp.raise_for_status()
                parser = SCRAPE_PARSERS[source_name]
                items = parser(resp.content, url)
                all_headlines.extend(items)
                logger.info("Scraped %d headlines from %s", len(items), source_name)
                any_success = True
            except Exception as e:
                logger.warning("Scraping failed for %s: %s", source_name, e)

        return all_headlines, not any_success

    def collect_news(config: Config) -> dict:
        """Collect financial headlines from NewsAPI and scraped sources.
        Returns dict with 'headlines' (max 35), failure flags, 'count', 'source_used'.
        Never raises — on failure returns partial dict with error flags.
        """
        conn = get_connection(config.db_path)
        cached = _get_cached(conn)
        if cached is not None:
            logger.debug("Cache hit for news headlines (%d items)", len(cached))
            conn.close()
            return {"headlines": cached, "count": len(cached), "newsapi_failed": False,
                    "scrape_failed": False, "partial": False, "source_used": CACHE_SOURCE}

        headlines: list[dict] = []
        newsapi_failed = False
        scrape_failed = False

        # Layer 1: NewsAPI
        if config.newsapi_key:
            try:
                api_headlines = _fetch_newsapi(config.newsapi_key)
                headlines.extend(api_headlines)
                logger.info("Fetched %d headlines from NewsAPI", len(api_headlines))
            except Exception as e:
                logger.error("NewsAPI fetch failed: %s", e)
                newsapi_failed = True
        else:
            logger.warning("NEWSAPI_KEY not configured — skipping NewsAPI")
            newsapi_failed = True

        # Layer 2: BS4 scraping
        scraped, scrape_failed = _scrape_all()
        headlines.extend(scraped)

        # Deduplicate by URL
        seen_urls: set[str] = set()
        deduped: list[dict] = []
        for h in headlines:
            if h["url"] not in seen_urls:
                seen_urls.add(h["url"])
                deduped.append(h)

        # Cap at MAX_HEADLINES
        final = deduped[:MAX_HEADLINES]

        _upsert_cache(conn, final)
        conn.close()

        return {
            "headlines": final,
            "count": len(final),
            "newsapi_failed": newsapi_failed,
            "scrape_failed": scrape_failed,
            "partial": newsapi_failed or scrape_failed,
            "source_used": CACHE_SOURCE,
        }
    ```

## Trust Boundaries

| Boundary | Description |
|----------|-------------|
| collector→NewsAPI | External HTTPS with API key; key must not be logged |
| collector→scrape targets | External HTTP; response HTML untrusted |
| scraper→BS4 | HTML parsing; no script execution |
| collector→SQLite | Local write; parameterised queries |

## STRIDE Threat Register

| Threat ID | Category | Component | Disposition | Mitigation Plan |
|-----------|----------|-----------|-------------|-----------------|
| T-02-17 | Information Disclosure | _fetch_newsapi | mitigate | apiKey sent as query param to HTTPS; logger.error logs exception message only, never the key value |
| T-02-18 | Tampering | Scraped HTML content | accept | BS4 parses HTML as data only — no script execution, no eval; extracted text stored as strings |
| T-02-19 | Denial of Service | Scraping targets | mitigate | SCRAPE_SLEEP_SECONDS=1.0 between domain requests; User-Agent header set to avoid immediate bot blocks |
| T-02-20 | Spoofing | Scraped headline titles/URLs | accept | Headlines treated as informational display data — no authentication or financial decisions based on URLs |
| T-02-21 | Tampering | _upsert_cache | mitigate | Parameterised sqlite3 query — headline data JSON-serialised before storage |
